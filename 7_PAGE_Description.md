# PAGE Description

To design an intelligent agent, we define its **PAGE** descriptor:

1.  **P**ercepts (What the agent sees/senses)
2.  **A**ctions (What the agent can do)
3.  **G**oals (What the agent tries to achieve)
4.  **E**nvironment (Where the agent operates)

---

## Detailed PAGE Examples

### 1. Automated Taxi Driver
| Feature | Description |
| :--- | :--- |
| **Percepts** | Cameras (visual), Speedometer, GPS, Odometer, Sonar. |
| **Actions** | Steer, Accelerate, Brake, Signal, Honk. |
| **Goals** | Safe trip, Fast transport, Legal driving, Passenger comfort. |
| **Environment** | Urban roads, Highways, Pedestrians, Traffic, Weather. |

### 2. Medical Diagnosis System
| Feature | Description |
| :--- | :--- |
| **Percepts** | Symptoms (patient input), Lab results, Medical history. |
| **Actions** | Ask questions, Order tests, Propose diagnosis, Treat. |
| **Goals** | Healthy patient, Correct diagnosis, Minimize cost/pain. |
| **Environment** | Hospital, Patient interaction, Digital medical records. |

### 3. Part-Picking Robot
| Feature | Description |
| :--- | :--- |
| **Percepts** | Pixels (Camera), Joint angles, Touch sensors. |
| **Actions** | Move arm, Open/Close gripper. |
| **Goals** | Place parts in correct bins, Fast operation. |
| **Environment** | Factory conveyor belt, Bins. |

### 4. Interactive English Tutor
| Feature | Description |
| :--- | :--- |
| **Percepts** | Student's typed words/answers. |
| **Actions** | Display exercises, Give feedback, Show corrections. |
| **Goals** | Student learns English, High test scores. |
| **Environment** | Computer screen, Keyboard, Student. |
